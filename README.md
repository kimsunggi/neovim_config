# `neovim` and `dein.vim` setup
***
### 配置先ディレクトリ  
**init.lua** と **dein.toml**を下記ディレクトリに配置  
`~/.config/nvim/`
***
### neovim
1. インストール  
`brew install neovim`

1. バージョン確認  
`nvim -v`

***
### dein.vim  
1. インストーラーダウンロード  
`curl https://raw.githubusercontent.com/Shougo/dein.vim/master/bin/installer.sh > installer.sh`
1. インストール  
`sh ./installer.sh ~/.cache/dein`
***
